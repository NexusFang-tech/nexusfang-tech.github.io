# MATT KEITH — Systems & Security Portfolio

**Live:** [nexusfang-tech.github.io](https://nexusfang-tech.github.io)

Cybersecurity and systems engineering portfolio built as a single-file HTML application with a vaporwave/synthwave aesthetic. Features an interactive hero with Canvas-rendered retrowave grid, scroll-triggered reveal animations, categorized project gallery with filtering, timeline experience section, and a fully functional embedded terminal with 15+ Easter eggs.

## Features

- **Retrowave hero** — Canvas-rendered perspective grid, animated sun with scan lines, and starfield
- **Category-filtered project gallery** — 25+ projects filterable across Interactive Tools, Security, IT Ops & Infra, Internal Tools, and Personal
- **Embedded terminal** — Full command system (`whoami`, `skills`, `projects`, `experience`, `certs`, `contact`) with ASCII banner, command history, Tab autocomplete, matrix protocol, and simulated nmap scan
- **Konami code** — Unlocks a playable SynthStrike mini rhythm game
- **Custom cursor** — Reactive cursor with hover scaling and color shifts
- **CRT scanline overlay** — Subtle post-processing effect across the entire page

## Tech Stack

`Vanilla HTML` · `CSS` · `JavaScript` · `HTML Canvas` · `Google Fonts` · `GitHub Pages`

Zero dependencies. No build step. Single HTML file.
